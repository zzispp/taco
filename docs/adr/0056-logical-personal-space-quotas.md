# Enforce logical quotas per personal space

Every Personal Asset Space has a logical Space Quota from a required 20 GiB system default or an explicit per-space override. Active files, Trashed Assets, and Quota Reservations all count against that limit. Upload Session creation atomically acquires a reservation for the declared file size, completion converts it into logical asset usage, and cancellation, expiry, or Permanent Deletion releases capacity. Global Stored Object deduplication does not discount a user's quota, so capacity rights remain predictable and concurrent uploads cannot oversubscribe the same space. Administrators may lower a quota below committed usage; this creates an explicit Over-Quota Space that retains read and cleanup operations but rejects new reservations until usage falls within the limit.
