# Bound inline preview formats

Personal Asset Spaces accept any non-empty file within the configured size limit, while the File Management API provides authenticated Inline Preview only for raster images, PDF, plain text and common text-based formats, plus browser-native audio and video with byte-range reads. Bounded thumbnails are generated only for PNG, JPEG, WebP, and the first GIF frame; images above 40 million decoded pixels remain storable but do not decode for thumbnail or preview. Image sources are counted before temporary staging and cannot exceed 32 MiB; a process runs at most two image validation or thumbnail jobs concurrently. Text previews read at most 1 MiB, escape rendered content, and mark larger previews as truncated. Office documents, archives, executables, HTML, scripts, unknown formats, and SVG remain Download-Only Assets served with attachment disposition. This separates storage from rendering capability, avoids an Office conversion service, and prevents same-origin script exposure.
